가설검정

  1. 귀무가설 / 대립가설 설정
  2. 데이터 타입에 맞는 가설검정 기법 사용
  3. P.value 계산 (귀무가설이 참일 확률)
  4. 유의수준 (5%)과 비교 후, 결론 도출
    (0.05 > p.value: 귀무가설 기각o) /
    (0.05 < p.value: 귀무가설 기각x)

단일 변수 검정

- 연속형: 정규성 검정 / One Sample T-test (1표본 평균 검정)

- 범주형 : 비율검정

다 변수 검정

- $X$: 범주형 / $Y$: 연속형: Two Sample T-test (2표본 평균검정) / Paired Test (대응 표본 검정)

- $X$: 범주형 / $Y$: 범주형: Chi Square Test

- $X$: 연속형 / $Y$: 연속형 $\to$ 상관분석 또는 회귀분석

- $X$: 연속형 / $Y$: 범주형 $\to$ (분류) Logistic Regression
